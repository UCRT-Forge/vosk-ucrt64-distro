# Vosk-UCRT64 Portable Distribution

Professional binary distribution of **Vosk-API** optimized for **Windows UCRT64**.

## Overview
This project provides a clean, portable build of Vosk for the modern **MSYS2 UCRT64** toolchain.

## Installation

### Python
Download the `.whl` from Releases and install:

```bash
pip install vosk-0.3.47-py3-none-any.whl
```
