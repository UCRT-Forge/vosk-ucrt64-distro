Security Policy / Политика безопасности
🇬🇧 English Version
1. Supported Versions
We provide security updates only for the latest version of vosk-ucrt64-distro available on the main branch.

2. Reporting a Vulnerability
Do not open a public Issue for security-related bugs.
If you discover a potential security vulnerability, please report it via:

GitHub Private Vulnerability Reporting: Use the "Security" tab in this repository.

Direct Email: Contact mkondr75 via the email associated with your GitHub account.

We aim to acknowledge your report within 48 hours and provide a fix as soon as possible.

🇷🇺 Русская версия
1. Поддерживаемые версии
Мы выпускаем обновления безопасности только для последней версии vosk-ucrt64-distro, доступной в ветке main.

2. Сообщение об уязвимости
Пожалуйста, не создавайте публичные Issues для багов, связанных с безопасностью.
Если вы обнаружили потенциальную уязвимость, сообщите о ней одним из способов:

GitHub Private Vulnerability Reporting: Используйте вкладку "Security" в этом репозитории.

Прямой Email: Свяжитесь с mkondr75 по адресу, указанному в профиле.

Мы постараемся подтвердить получение отчёта в течение 48 часов и подготовить исправление в кратчайшие сроки.
