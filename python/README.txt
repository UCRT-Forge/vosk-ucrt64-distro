ИНСТРУКЦИЯ ПО МОДЕЛЯМ ДЛЯ PYTHON:
1. Скачайте нужную модель с официального сайта AlphaCephei (например, vosk-model-small-ru-0.22).
2. Распакуйте её в папку вашего проекта (например, в подпапку models/).
3. В коде всегда указывайте явный путь:
   model = vosk.Model("models/vosk-model-small-ru-0.22")
4. Избегайте использования lang="en-us", чтобы исключить неконтролируемые загрузки из интернета.
