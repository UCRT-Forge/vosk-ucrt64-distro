Contributing to UCRT-Forge / Участие в проекте
Welcome! Thank you for your interest in the vosk-ucrt64-distro project. This document outlines the rules for contributing.

Добро пожаловать! Спасибо за интерес к проекту. Этот документ описывает правила участия в разработке.

🇬🇧 English Version
1. Reporting Issues
Search through existing Issues to ensure your problem hasn't been reported yet.

If it is a new bug, please provide:

Your Windows version.

Your compiler version (e.g., gcc --version in MSYS2).

A minimal code snippet or a link to a repository where the issue occurs.

2. Pull Requests (PR)
Fork the repository and create your branch from main.

All binaries (DLLs) must be built strictly using the MSYS2 UCRT64 toolchain.

Ensure your changes do not break existing test examples.

Open a Pull Request with a clear description of the changes.

🇷🇺 Русская версия
1. Сообщение об ошибках
Проверьте существующие Issues, чтобы убедиться, что проблема еще не обсуждалась.

Если вы нашли новый баг, пожалуйста, укажите:

Вашу версию Windows.

Версию вашего компилятора (например, gcc --version в среде MSYS2).

Минимальный фрагмент кода или ссылку на репозиторий, где воспроизводится ошибка.
